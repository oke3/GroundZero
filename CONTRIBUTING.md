# Contributing to GroundZero

Thank you for your interest in helping us build the definitive technology news reference directory!

## 🛠️ How to Contribute

### 1. Adding a New News Outlet
If you find a high-quality news source that is missing:
1. Fork this repository.
2. Open `directory.md`.
3. Find the appropriate vertical section.
4. Add the outlet using the following format:
   `| Outlet Name | URL | Focus |`
5. Submit a Pull Request with a brief description of why this outlet is a valuable addition.

### 2. Correcting Existing Information
If a URL is broken or a description is inaccurate:
1. Fork this repository.
2. Update the information in `directory.md`.
3. Submit a Pull Request.

## ⚖️ Contribution Criteria
To maintain the quality of the directory, we look for sources that:
- **Reliability**: Have a track record of accurate reporting.
- **Focus**: Provide deep coverage of their designated technology vertical.
- **Consistency**: Update their content regularly.

## 💬 Communication
For major changes or suggestions for new verticals, please open an Issue first to discuss the proposal.
